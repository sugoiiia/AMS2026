# Manual Maze Navigation

## Challenge
Navigate the maze using the provided controller. Worth **20 points**.

## Notes
- Uses starter code from AlbertaLoop (`ams_2026_starter_code.ino`) as base
- Joystick X → left/right, Joystick Y → forward/backward
- Tune the deadzone values (470/550) if the robot drifts when joystick is centered
